# AI Dev Tools presentation slides

- Using [Marp] (compatible with DeckSet)

## Usage

```bash
# render html and watch
marp --watch index.md

# or render a pdf
marp --pdf --allow-local-files index.md
```

[Marp]: https://marp.app/
